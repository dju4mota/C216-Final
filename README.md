## Backend
Para rodar o backend precisa ter o docker instalado. Com ele instalado e funcionando, você pode rodar os comandos: 

```docker-compose up --build ```



### Entidades:

Pet: 
* nome 
* animal 
* raça 
* idade 
* adotavel 
* sociavel
